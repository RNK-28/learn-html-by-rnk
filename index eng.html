Complete HTML

What is HTML?

HTML stands for HyperText Markup Language and it is used to create websites. A webpage is basically divided into 3 parts:

1. HTML (HyperText Markup Language)

2. CSS (Cascading Style Sheets)

3. JS (JavaScript)

For Example: You built a house that currently has no paint or beautiful furniture. We can call this HTML. Then, when we paint it and add beautiful furniture, we are styling the house, which is called CSS. After that, we add things like a microwave which needs a function to work. This is called JS (JavaScript).

For now, we will only learn HTML, nothing else, OK?

Structure of HTML

To understand HTML easily, you don't need to create its structure repeatedly or memorize it. You just need to copy-paste this structure and start writing your code inside the <body></body> tag.

The full structure is given below:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

Look, there's no need to be scared seeing all this. You don't have to memorize anything. Just copy-paste this and start writing. In the title tag, you can put the name of your website, and then start writing your code inside the body tag.

What are Tags?

First, understand how tags are written. They are written like this: <tag_name>. Every tag has an ending tag like </tag_name>, but some don't. We will learn about those later.

For now, let's learn about two tags: <p> and <h1>. First, let's look at the <p> tag. It is used to print a paragraph on the screen. The <h1> tag is for a heading, just like we give a heading before writing a paragraph in an essay. But there is not just one <h1> tag; there are 6 more similar tags.

h1 (Most important heading)

h2 (A slightly less important heading)

h3 (Normal heading)

h4 (This won't be as important)

h5 (This will be even less important)

h6 (This won't be important at all)

An example code for this is given below


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1>Heading No.1</h1>
    <h2>Heading No.2</h2>
    <h3>Heading No.3</h3>
    <h4>Heading No.4</h4>
    <h5>Heading No.5</h5>
    <h6>Heading No.6</h6>
</body>
</html>

Copy this and run it in any HTML runner.


Now, let's learn about the <p> tag. It stands for paragraph and prints a paragraph on the screen. Its example code is below.


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1>Define HTML</h1>
    <p>HTML is the code that defines the structure and content of a web page.</p>
    
</body>
</html>
