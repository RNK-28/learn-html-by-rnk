Structure Of HTML

HTML ko agar asani se samajna hai to ye samjo ke ap ko is ka structure bar bar bnane ki zarorat nhi hai na hi zabani yad karne ki zarorat bas ye structure copy paste karna hai or <body></body> is tag ke andar apna code likhna start kar de

Niche full structure dya gya hai


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

Dekho ye sab dekh kar darne ki zarorat nhi hai tumhe kuch bhi zabani yaad karne ki zarorat nhi hai to bas is ko copy paste karo or likhna start karo title wale tag main apni website ka name dal dena phir body tag ke andar apna code likhna start karna

Tags Kya Hai?

Sab se pehle ye samjo ke tags likhte kaise hai wo ese likhte hai <tag_name> har tag ka ek ending tag zaror hota hai kuch aisa </tag_name> par kuch kuch ke nhi hote wo ham agge chal ke parhe ge.

Filhal abhi ham 2 tags ke bare main jante hai ek hai <p> or <h1> sab se pehle ham p wale tag ko dekhte hai ke ye kya kam karta hai ye kam karta hai ke screen main paragraph ko print karwata hai or h1 tag heading hoti hai jese ham paper karte hai pehle heading dete hai phir paragraph likhte hai ye wese hi hota hai lekin h1 tag ek hi nhi hota ese 6 or tag hote hai.

h1 (Most important heading)

h2 (Os se thory important heading)

h3 (Normal heading)

h4 (ye itni important nhi hogi)

h5 (ye to os se bhi zyada important nhi hogi)

h6 (ye important hi nhi hogi)

Is ka ek example code niche dya gya hai


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1>Heading No.1</h1>
	<h2>Heading No.2</h2>
	<h3>Heading No.3</h3>
	<h4>Heading No.4</h4>
	<h5>Heading No.5</h5>
	<h6>Heading No.6</h6>
</body>
</html>
